# Fix The Roof ICI

## Overview
A static HTML advocacy website for a small trading card business dealing with an unresolved roof leak issue with their landlord. The site documents their situation and calls for action.

## Project Structure
- `index.html` - Single-page static website with inline CSS
- `pressure-campaign-kit-final.md` - Supporting documentation

## Technical Details
- **Type**: Static HTML website (no build system)
- **Server**: Python http.server for development
- **Port**: 5000
- **Deployment**: Static deployment

## Running the Project
The site runs using Python's built-in HTTP server:
```
python -m http.server 5000 --bind 0.0.0.0
```
